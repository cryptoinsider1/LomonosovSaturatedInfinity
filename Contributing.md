# Contributing Guidelines

Спасибо за интерес к проекту **LomonosovSaturatedInfinity**.

## Как помочь
- Сообщайте об ошибках через Issues.
- Предлагайте улучшения через Pull Requests.
- Сохраняйте академичность и чистоту формулировок.

## Стиль кода
- Python 3.9+
- Использовать `numpy`
- Докстринги в стиле Google или NumPy.

## Этическая заметка
Проект задуман как философско-научный артефакт.
Пожалуйста, сохраняйте уважение к контексту и источникам.
